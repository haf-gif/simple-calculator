# 🚀 DevOps Professional Glassmorphism Calculator

A modern, production-grade web calculator built to demonstrate essential DevOps and Frontend skills.

## ✨ Features
- **Modern UI:** Advanced Glassmorphism design with a responsive layout.
- **Scientific Functions:** Support for Square Root and Pi.
- **History Log:** Persistent calculation history display.

## 🛠️ DevOps & Production Features (Hire Me! 😉)
- **Containerization:** Built with a highly secure and lightweight **Nginx Alpine** base image.
- **Security:** Nginx is configured to run as a **non-root user** and includes **custom security headers** (`X-Frame-Options`, `X-Content-Type-Options`).
- **Production-Ready Configuration:** Custom `nginx.conf` instead of default settings.
- **Infrastructure as Code (IaC):** Single-command deployment using **Docker Compose** with integrated container naming, port mapping, and basic resource logging configurations.
